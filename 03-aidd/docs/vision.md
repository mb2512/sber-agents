# Техническое видение проекта LLM-ассистента

## Технологии

- **Python** — основной язык разработки.
- **uv** — для управления зависимостями.
- **openai client** — для подключения к LLM через провайдера Openrouter (через OpenAI-совместимый API).
- **make** — для автоматизации сборки и типовых операций.
- **aiogram** — для интеграции с Telegram Bot API и обработки сообщений через polling-метод.

Используются только обязательные технологии, необходимые для запуска минимально жизнеспособного прототипа.
